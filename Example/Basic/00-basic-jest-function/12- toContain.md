# Jest `toContain()` Matcher

## คำอธิบาย

ใน Jest, `toContain()` เป็น matcher ที่ใช้ตรวจสอบว่า **ค่าเฉพาะเจาะจง** นั้นมีอยู่ใน **อาร์เรย์** หรือ **สตริง** ที่ทดสอบหรือไม่

- เมื่อทดสอบกับอาร์เรย์: ตรวจสอบว่าค่าที่ต้องการมีอยู่ในอาร์เรย์นั้นหรือไม่
- เมื่อทดสอบกับสตริง: ตรวจสอบว่าสตริงมีส่วนของข้อความที่กำหนดไว้หรือไม่

## การใช้งาน

1. **ตรวจสอบการมีอยู่ในอาร์เรย์**

   ```javascript
   test("ตรวจสอบการมีอยู่ของค่าในอาร์เรย์", () => {
     const fruits = ["apple", "banana", "grape"];
     expect(fruits).toContain("banana");
   });



   test("ตรวจสอบการมีอยู่ของข้อความในสตริง", () => {
     const sentence = "JavaScript testing with Jest";
     expect(sentence).toContain("testing");
   });
   ```
